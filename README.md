# scoop-fernsicht

Scoop bucket for [Fernsicht](https://github.com/MuteJester/Fernsicht) — real-time remote progress monitoring over peer-to-peer WebRTC.

## Install

```powershell
scoop bucket add fernsicht https://github.com/MuteJester/scoop-fernsicht
scoop install fernsicht
```

## Verify

```powershell
fernsicht --version
```

## Updating

Manifest files are published automatically by the
[release workflow](https://github.com/MuteJester/Fernsicht/tree/main/.github/workflows)
in the main Fernsicht repo. Don't edit `fernsicht.json` by hand —
it's regenerated every release.
